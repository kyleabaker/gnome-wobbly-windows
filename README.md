# GNOME Wobbly Windows effect for GNOME Shell

GNOME Wobbly Windows effect for GNOME Shell. This extension mimics the original Compiz plugin engine to render the Wobby Windows feature.

This implementation introduces optimizations and bugfixes to the original implementation by [Mauro Pepe](https://github.com/hermes83/compiz-windows-effect).

Optimizations and bugfixes:

- Optimized code for increased performance.
- Reduced big-o complexity of multiple functions.
- Reduced code duplication.
- Reduced code complexity.
- Reduced memory usage.
- Added null checks for stabilization
- Modernized code structure and syntax.
- Stability fixes and bugfixes.
- Additional stability fixes for previously unchecked edge cases and undefined behavior.

## Prerequisite

Note: This extension does not require any additional or external libraries for functionality.

## Installation

You can install this extension by visiting [the GNOME Shell Extensions page](https://extensions.gnome.org/extension/) for this extension.

[<img src="assets/get-it-on-ego.png" height="100">](https://extensions.gnome.org/extension/)

## Troubleshooting

In case of update error please restart GNOME Shell via:

1. Xorg: press ALT+F2 then write r and press enter.
2. Wayland: end the session by logging out and log in again.
